# Smart Campus Resource Booking & Utilization Application

## Description
This project is a web-based system to manage booking and utilization of campus resources like classrooms, labs, and equipment.

## Features
- User login & registration
- Resource booking system
- Admin management
- Availability tracking

## Technologies Used
- Frontend: HTML, CSS, JS
- Backend: Django / Node.js
- Database: MySQL

## How to Run
1. Start backend server
2. Connect to MySQL database
3. Run frontend
